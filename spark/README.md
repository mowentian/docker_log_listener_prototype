
## deploy

```sh
docker build -t pp-spark .
```

please refer: [sequenceiq/docker-spark](https://github.com/sequenceiq/docker-spark)

## run

```sh
docker run -it -p 9999:9999 -h sandbox pp-spark bash
```

spark-submit \
